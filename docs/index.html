Develop an analytics Dashboard with React

Prerequisites for install

  • You need Git to clone the repository

  • You need Node (v16.7.0) to run the npm commands

Dependencies

  • React

  • React-router-dom

$ npm install react-router-dom

  • Recharts

$ npm install recharts

  • Axios

$ npm install axios

  • Prop-types

$ npm i prop-types

  • Recommended text editor: Visual Studio Code

Installing and launching Back-End

  1. Clone the repository of SportSee back-end:

git clone https://github.com/Alexlemars/P12_Backend-API.git

  1. Inside this back-end repository, install dependencies:

npm install

  1. Launch back-end on port 3000 (default port):

npm run start

Installing and launching Front-End

  1. Clone the repository of SportSee front-end:

git clone https://github.com/Alexlemars/boucher_alexis_12_25012022.git

  1. Inside this front-end repository, install dependencies:

npm install

  1. Launch front-end on port 3001:

npm start

  1. Front-end is now rendered at URL http://localhost:3001.

  2. URL availables :

http://localhost:3001/user/12

http://localhost:3001/user/18

N.B.:

  • SportSee API only contains data for users with id '12' and '18'.
